# Электроника Онлайн (Electronics Online)

## Overview
A static HTML/CSS/JavaScript website for an electronics store. The site is in Russian and features a homepage, product catalog, and contact page.

## Project Structure
- `index.html` - Homepage with hero section
- `catalog.html` - Product catalog page
- `contact.html` - Contact information page
- `styles.css` - Stylesheet for the site
- `script.js` - JavaScript for product purchase alerts
- `server.py` - Simple Python HTTP server for development

## Development
Run the web server workflow to start the development server on port 5000.

## Deployment
Configured for static deployment. The entire project root is served as static files.
